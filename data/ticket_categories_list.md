فئات التذاكر في NileTel

يا فندم، اختيار الـ category الصحيح مهم جداً عشان التذكرة تروح للـ team المناسب بسرعة.

الفئات الرئيسية:
- Network Issue → 5G Slow Speed / 4G Drop / FTTH Outage / ADSL Problem
- Billing → Overcharge / Refund Request / Double Billing / Roaming Charges
- Service Request → New Line Activation / Upgrade Package / Cancellation / Prepaid to Postpaid Migration
- Technical → Device Not Connecting / App Error / Account Lock / ONT Issue
- VIP / Retention → Special handling for high-value customers

نصيحة: لو المشكلة فيها أكتر من فئة، اختار الأهم ودوّن التفاصيل في الـ description.
